# ksh - shell
Created:  [[2022-07-23]]
Tags: #fleeting 

---
`korn shell` is a superset version of [[sh - shell]]
- allows built-in support for arithmetic operations
- supports scripts made in
    - Bourne shell
    - C Shell
        - while offering string, array and function manipulation 

`/bin/ksh`
`#` for the root user
`$` for the non-root users
