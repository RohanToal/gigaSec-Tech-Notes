# Selection in vim
Created:  [[2022-07-08]]
Tags: #permanent 

---
#### Select All
**ggVG**
`gg` jumps to beginning of file
`VG` selects from current cursor to end of document.

#### Navigation of selection
`#` navigate similar words of current word cursor is at
`$` go to end of the line
