# printf in C
Created:  [[2022-08-14]]
Tags: #fleeting 

---
Uses: `#include <stdio.h>`

The `%s` format in printf expects argument passed to be characters that ends with `\0`.
Meaning its possible to use `%s` with value being only `\0`.
The result will be `(null)`
