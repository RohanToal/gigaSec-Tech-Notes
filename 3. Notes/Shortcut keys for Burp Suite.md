# Shortcut keys for Burp Suite
Created:  [[2022-07-08]]
Tags: #fleeting 

---
`Ctrl + Shift + D`  
Switch to the Dashboard  


`Ctrl + Shift + T`  
Switch to the Target tab  


`Ctrl + Shift + P   `
Switch to the Proxy tab  


`Ctrl + Shift + I   `
Switch to the Intruder tab  


`Ctrl + Shift + R   `
Switch to the Repeater tab
