# Subnetting use IP address in three ways
Created:  [[2022-04-23]]
Tags: #permanent 

---
Network Address, 
- this determines the starting point of a network.


Host Address, 
- identifies the devices in a network


Default Gateway, 
- used when we want to SEND data OUTSIDE of our `Network Address`
