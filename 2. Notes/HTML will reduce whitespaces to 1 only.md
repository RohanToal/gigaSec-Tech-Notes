# HTML will reduce whitespaces to 1 only
Created:  [[2022-09-12]]
Tags: #fleeting 

---
HTML parser will always reduce whitespaces to 1 only
For instance
Is the same as this below
```HTML
<p>Dogs                  are
                   silly</p>
```
Will output into
-> `Dogs are silly`
