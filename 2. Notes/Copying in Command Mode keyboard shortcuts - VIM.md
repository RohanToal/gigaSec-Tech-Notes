# Copying in Vim
Created:  [[2022-07-09]]
Tags: #fleeting 

---
`yank` is copying in vim

`y` copy single character
You can go `visual mode` and select texts
then press `y` to copy selected text

`yw` yanks one word, 

`count yy`
`yy` for copy single line

Copy two lines
`2yy`


Copy two end of the line
`y$`
