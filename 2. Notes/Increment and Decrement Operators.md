## Rules on usage of Increment/Decrement Operators

Can ONLY be applied to variables
Can be used as a [[subscript]] -> `s[j++]`

**Aren't Allowed**
[[Expression]]s ARE NOT allowed to have Increment/Decrement Operators
    Like -> `(i + j)++`
