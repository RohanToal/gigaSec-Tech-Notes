Created:  [[2022-07-26]]

THREE WAYS TO DEFINE DICTIONARY

 
1st Way
```python
d = {
    'key': 'value', 
    'key': 'value'
}
```


2rd Way (If `key-value` pair are all strings)
```python
d = dict(
    key = 'value',
    key = 'value'
)
```
