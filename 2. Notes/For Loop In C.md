# For Loop

```C
for (expr1; expr2; expr3)`
    statement
```

## Can omit [[expression]]s in for loop
```C
for (;;)
```

When `expr2` is omitted, it is taken as permanently true.
If `expr1` or `expr3` is omitted, it is simply dropped from the expansion
