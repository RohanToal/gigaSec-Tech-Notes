# Character Constant is used to convert char to int-ascii
Created:  [[2022-08-15]]
Tags: #fleeting 

---
A character surrounded by `''` like `'A'` is called **Character Constant** 
It makes it possible to convert character into ASCII
```C
int num = 'A';
printf("%i", num);
```
